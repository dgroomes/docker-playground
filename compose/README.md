# compose

A simple Docker Compose example.

## Instructions

* `docker-compose up`
* You should see something like:
  ```
  Recreating compose_my-busybox_1 ... done
  Attaching to compose_my-busybox_1
  my-busybox_1  | hello world!
  compose_my-busybox_1 exited with code 0
  ```
